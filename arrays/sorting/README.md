# Sorting

```
| Algorithm      | time        | expected time |
|----------------+-------------+---------------|
| Insertion sort | O(n^2)      |               |
| Merge sort     | O(n*log(n)) |               |
| Bubble sort    | O(n^2)      |               |
| Quick sort     | O(n^2)      | O(n*log(n))   |
```
